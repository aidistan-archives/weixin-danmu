安装插件

对于 Firefox 浏览器

  • 从火狐扩展商店安装（推荐，插件能自动更新）
  • 从插件官网直接安装

对于 Chrome 浏览器

  • 从谷歌网上应用商店安装（推荐，插件能自动更新）
  • 从插件官网下载压缩包，解压后将文件夹直接拖入 Chrome 的扩展页
即将以 Chrome 插件方式支持 Edge 浏览器

基本使用

  1. 安装插件
  2. 在浏览器的任意一个标签页中登陆网页版微信
  3. 打开要捕获消息的群或好友的对话框
  4. 切换到要显示弹幕的任意一个网页
  5. 在手机微信上向所选的好友或群发消息
  6. 弹幕出现！

详细说明

使用工具栏按钮

  • 未侦测到对话框时按钮呈灰色，点击按钮将打开网页微信页面
  • 打开对话框后按钮会自动点亮，点击按钮将打开弹幕助手页面

修改默认设置

打开浏览器插件页 ( Ctrl + Shift + A， 木有快捷键)，点击微信弹幕的设置按钮进入附加组件详情页面，从中可以：

  • 开启/关闭显示所有消息
  • 调整弹幕的默认大小
  • 开启/关闭好友昵称显示
  • 开启/关闭微信标签固定
  • 开启/关闭消息通知显示
  • 开启/关闭弹幕测试模式
通过配置是否“显示所有消息”，微信弹幕既可以在活动中使用固定群聊作为弹幕源而不会泄漏私人消息，也可以在自己上网时监听来自所有人的微信消息。

发射弹幕

微信发送的任意文字、表情、图片，默认会以固定大小、随机颜色、随机位置显示在屏幕上。（部分表情在网页微信上不会显示，因此无法做成弹幕）

目前支持自定义弹幕的颜色、大小和位置，格式为 ：字，具体说明如下：

  • 冒号同时支持中文全角符号和英文半角符号
  • 冒号和魔力字符（效果示意中的灰色部分）不会显示在弹幕中
  • 属性可以按照任意顺序设置，但必须放在消息的开头
  • 设置字幕大小的效果可以叠加，即“:大:大:大”的效果是":大"的三倍（但大小有上下限）

常见问题

问：只能在浏览器的标签页上显示弹幕，有什么用途？
答：随着浏览器功能的逐步增强，浏览器版的弹幕已可以满足绝大多数的需求：

  • 传统幻灯片可以导出PDF，用浏览器直接播放
  • 推荐使用网页幻灯片，Shower 和 deck.js 都是不错的引擎
  • HTML5已原生支持视频的播放，可用 vidio 标签把视频嵌在网页里，也可直接把文件拖进浏览器播放。建议使用MP4格式，转码可以用 MediaCoder，最好用的开源转码软件，没有之一（全屏问题请 移步至此 ）
  • 什么！你问音频怎么办？……别闹了
如果你有特殊需求，自己尝试使用但未成功，请在 讨论区 中详细说明情况，我会尽量帮忙。

问：视频一全屏，弹幕就不见了，怎么破？
答：受浏览器和技术限制，弹幕无法支持 HTML5 视频 和 Flash 视频 的全屏功能。目前我暂时也不知道 Flash 视频怎么办，HTML5 视频有替代的解决方案：

  • 直接用浏览器播放：可以使用浏览器的全屏功能达到相同效果（请按F11）
  • 视频嵌在网页里：请将 video 设置成合理的大小，其余同上

问：弹幕助手是什么，有什么用途？
答：基于 Shower 制作的弹幕助手，在作为 网页幻灯片 的一个示例的同时，还提供了 群聊入口 和 使用帮助，是活动组织者的有力工具。

  • 群聊入口二维码：需根据实际情况，将本地二维码图片拖进方框
  • 使用帮助二维码：扫描后会跳转至当前页面的弹幕发射说明部分

问：QQ 表情的显示有些奇怪，怎么回事？
答：网页版微信内置的 QQ 表情像素过低... 建议改为使用 Emoji 表情。

问：Chrome 版插件显示弹幕时，总有些延迟是怎么回事？
答：这是 Chrome 自身的限制。出来混，总是要还的，我只能说这么多。
